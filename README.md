# Integración de MCP en GitHub Copilot para desarrolladores

Este es el repositorio del curso de LinkedIn Learning `[Integración de MCP en GitHub Copilot para desarrolladores]`. El curso completo está disponible en [LinkedIn Learning][lil-course-url].

![Nombre completo del curso][lil-thumbnail-url] 


DESCRIPCIÓN DEL CURSO
Lleva GitHub Copilot al siguiente nivel integrando Model Context Protocol (MCP) en tus flujos de desarrollo. Este curso te guía paso a paso para crear y personalizar MCPs que amplían las capacidades de Copilot, conectando tus herramientas, automatizaciones y recursos internos. A través de ejemplos prácticos, aprendes a diseñar, implementar y publicar MCPs que optimizan tareas repetitivas, mejoran la colaboración y te ofrecen un entorno de trabajo más eficiente y adaptado a tus necesidades, tanto en proyectos individuales como en equipos de desarrollo.

## Instrucciones

Este repositorio tiene ramas (branches) para cada uno de los vídeos del curso. Puedes usar el menú emergente de la rama en GitHub para cambiar a una rama específica y echar un vistazo al curso en esa etapa, o puedes añadir `/tree/nombre_de_la_rama` a la URL para ir a la rama a la que quieres acceder.

## Ramas

Las ramas están estructuradas para corresponder a los vídeos del curso. La convención de nomenclatura es Capítulo#_Vídeo#. Por ejemplo, la rama denominada `02_03` corresponde al segundo capítulo y al tercer vídeo de ese capítulo. Algunas ramas tendrán un estado inicial y otro resuelto.  

## Instalación

1. Para utilizar estos archivos de ejercicios, debes tener descargado lo siguiente:
   - Nodejs
   - Visual Studio Code con Github Copilot
   - Claude (opcional)
