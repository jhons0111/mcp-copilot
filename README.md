# MCP Protocol Course

A comprehensive, hands-on course on the **Model Context Protocol (MCP)** — the open standard that bridges AI assistants and third-party applications.

---

## What You Will Learn

* What MCP is and the problem it solves
* Core architecture: servers, clients, tools, resources, and prompts
* Setting up a development environment (Node.js and Python)
* Building your first MCP server from scratch
* Defining tools that AI models can call
* Exposing data through resources
* Creating reusable prompt templates
* Connecting AI clients (Claude Desktop, Cursor, etc.) to your server
* Building real-world integrations end-to-end

## About MCP

The **Model Context Protocol** is an open standard created by Anthropic that allows AI models to securely connect to external tools, APIs, databases, and services. Think of it as a universal plug for AI — one protocol, any integration.

Official documentation: <https://modelcontextprotocol.io>
